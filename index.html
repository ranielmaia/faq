FAQs

O que é Ginseng?

O ginseng é uma planta medicinal com diversos benefícios para saúde, ele tem ação estimulante e revitalizante, sendo ótimo para quando se está muito cansado, estressado e precisando de um estímulo extra para continuar com as atividades diárias.

O que é Ginkgo biloba?

O Ginkgo biloba é uma planta medicinal ancestral da China que é bastante rica em flavonóides e terpenóides, tendo assim uma forte ação anti-inflamatória e antioxidante.

O que é Catuaba?

A Catuaba, também conhecida como Alecrim-do-campo, Catuaba-verdadeira, Catuabinha, Catuíba, Catuaba-pau, Caramuru ou Tatuaba, é uma planta medicinal muito utilizada para fazer remédios afrodisíacos para problemas de impotência masculina.

O que é Guaraná em Pó?

O pó de guaraná é feito a partir das sementes do guaraná, e trás benefícios como aumentar o estado de alerta e atenção, melhorar o humor e estimular a queima de gordura no corpo, sendo uma excelente opção para dar mais disposição para o treino e para dietas de emagrecimento.
